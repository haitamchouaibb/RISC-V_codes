# RISC-V Codes

Repository of exercises and projects in RISC-V Assembly for the **Computer Architecture** course at Politecnico di Torino.

This repository contains a collection of RISC-V Assembly programs developed as part of the Computer Architecture course. The exercises cover various aspects of Assembly programming, from array and matrix processing to compression algorithms.

## ğŸ—‚ï¸ Contents

The repository includes the following programs:

- **RLEencode.s** - Implementation of the Run-Length Encoding (RLE) compression algorithm
- **Ricostruzione255.s** - Value reconstruction with saturation at 255
- **calcolaPrezzi.s** - Price calculation and processing
- **calcolaSequenza.s** - Numerical sequence processing
- **contaMaggiori.s** - Count elements greater than a threshold in an array
- **contaUguali.s** - Count occurrences of equal values
- **maxExcursion.s** - Calculate maximum excursion in a dataset
- **sommaMatrice.s** - Matrix sum operations
- **tempesta.s** - Storm data simulation or processing
- **trovaMax.s** - Find maximum value in an array
- **trovaModa.s** - Calculate mode (most frequent value) in a dataset
